# Gallery retriever

This small script fetches image galleries from the web based on the provided URL. 
Creates folder in the specified location and fetches all images with their appropriate names and extensions.


## Usage

```bash
usage: get_gallery.py [-h] --url URL [--outputFolder OUTPUTFOLDER]

This script retrieves image galleries based on the provided URL.

optional arguments:
  -h, --help            show this help message and exit
  --url URL             Input URL pointing to a gallery or a photo
  --outputFolder OUTPUTFOLDER
                        Folder in which the gallery will be saved.
```
